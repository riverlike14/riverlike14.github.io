---
title: \[Baekjoon Online Judge - 12925\] Numbers
---

[문제](https://www.acmicpc.net/problem/12925)

다음 세 가지를 생각해야 한다.
1. $3 + \sqrt{5}$를 근으로 갖는 이차방정식 $x^2 - 6x + 4 = 0$의 다른 한 근 $3 - \sqrt{5}$을 생각.
1. 모든 자연수 $n$에 대하여 $0 < (3 - \sqrt{5})^n < 1$이 항상 성립.
1. $(3 + \sqrt{5})^n + (3 - \sqrt{5})^n$을 $1000$으로 나눈 나머지의 주기.

먼저, 음이 아닌 정수 $n$에 대해서 $s_n = (3 + \sqrt{5})^n + (3 - \sqrt{5})^n$으로 정의하자. 이때 다음이 성립한다.
- $s_0 = 2, s_1 = 6$.
- $s_{n+2} = 6s_{n+1} - 4s_n$.

위 사실들로부터 모든 음이 아닌 정수 $n$에 대하여 $(3 + \sqrt{5})^n + (3 - \sqrt{5})^n$이 자연수라는 사실을 알 수 있다.
한편, $(3 - \sqrt{5})^n$은 모든 자연수 $n$에 대하여 그 값이 $1$보다 작은 양수이다.
따라서 $(3 + \sqrt{5})^n$의 소수점 앞 마지막 세 정수는 $s_n$에서 $1$을 뺀 값의 마지막 세 정수와 같게 된다.

끝으로, 음이 아닌 정수 $n$에 대해 $s_n$을 $1000$으로 나눈 나머지를 $t_n$으로 정의하자.
이때 $t_0 = 2, t_1 = 6$ 이고 $s_n$과 마찬가지로 $t_{n+2} \equiv 6t_{n+1} - 4t_n \text{ mod 1000 }$이 성립한다.
그리고 $3$ 이상의 자연수 $n$에 대하여 $t_{n+100} = t_n$이 성립하는데, 이 사실은 직접 계산을 통해서 알 수 있다.

따라서 모든 자연수 $n$에 대하여 $t_n$을 알고 싶다면 $t_1, t_2, \cdots, t_{102}$를 구하는 것만으로 충분하다.

```c++
#include <iostream>
using namespace std;

int arr[103] = {2, 6};
int tn(int num);

int main() {
    int T; cin >> T;
    int N;
    for (int t = 1; t <= T; t++) {
        cin >> N;
        if (N >= 3)
            N = (N - 3) % 100 + 3;
        printf("Case #%d: %03d\n", t, (1000 + tn(N) - 1) % 1000);
    }
    return 0;
}

int tn(int num) {
    if (!arr[num])
        arr[num] = (4000 + 6*tn(num - 1) - 4*tn(num - 2)) % 1000;
    return arr[num];
}
```
